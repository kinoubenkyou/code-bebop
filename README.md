CODE BEBOP
================

Personal tech blog

Published at: [kinoubenkyou.github.io/code-bebop/](https://kinoubenkyou.github.io/code-bebop/)

Dependencies
------------

* python2
* pelican
* ghp-import

Usage
--------

Local:

```shell
pelican -lr
```

Publish on Github:

```shell
pelican -s publishconf.py
ghp-import -p output
```
